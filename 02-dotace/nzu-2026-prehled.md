# Nová zelená úsporám 2026+ — přehled

Program SFŽP / MŽP, nová etapa vyhlášena **28.5.2026**.

## Co je nového od 2026
- Pro většinu žadatelů místo přímé dotace **bezúročný úvěr** (úroky hradí fond).
- **Povinný renovační pas / energetický posudek.**
- Síť **certifikovaných energetických specialistů** (jejich zapojení povinné).
- **NZÚ Light** = přímá dotace pro nízkopříjmové / seniory.
