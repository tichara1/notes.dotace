# Rekonstrukce domu + dotace NZÚ 2026+

Poznámky a plánování rekonstrukce rodinného domu s dotací **Nová zelená
úsporám 2026+** (vyhlášena 28.5.2026). Repo se publikuje jako GitHub Pages.

## Jak to používat

- Píšeš do `.md` souborů ve složkách `01-` až `06-`. Každý už má kostru
  (nadpisy, checklisty, tabulky) — jen doplňuješ konkrétní poznámky.
- Hotovou položku v checklistu odškrtni: `- [x]`.
- Nový dokument/sken nahraj do `06-dokumenty/` a odkaž na něj.

## Obsah

1. [Přehled](01-prehled/) — cíl, rozsah, harmonogram
2. [Dotace](02-dotace/) — NZÚ 2026+, podklady, financování
3. [Specialisté](03-specialiste/) — energetický specialista, kontakty
4. [Realizace](04-realizace/) — firma vs subdodavatelé, poptávky, nabídky
5. [Rozpočet](05-rozpocet/) — náklady, dotace/úvěr, vlastní zdroje
6. [Dokumenty](06-dokumenty/) — deník jednání a dokumenty

## Publikace na GitHub Pages

1. Push do GitHubu.
2. Repo → **Settings → Pages** → Source: *Deploy from a branch* → `main` / `/ (root)`.
3. Web naběhne na `https://tichara1.github.io/notes.dotace/`.
